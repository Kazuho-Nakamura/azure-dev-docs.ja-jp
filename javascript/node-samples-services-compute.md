| | |
|---|---|
| **Azure Functions** ||

| **仮想マシンの管理** ||
| [Node.js による Azure 仮想マシン管理のサンプル](https://github.com/Azure-Samples/compute-node-manage-vm) | 仮想マシンの作成、リスト、再起動、削除の方法について説明します。 |
